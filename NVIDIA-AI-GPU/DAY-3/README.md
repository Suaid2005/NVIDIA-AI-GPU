# Day 3 - Introduction to Neural Networks and Deep Learning

## Overview

The third day focused on the fundamentals of Neural Networks and Deep Learning. The session covered the architecture of neural networks, activation functions, and practical implementations using Python, Scikit-Learn, and TensorFlow.

## Learning Objectives

* Understand the structure of a Neural Network.
* Learn the role of Input, Hidden, and Output layers.
* Explore activation functions and their importance.
* Implement basic neural network examples.
* Train deep learning models using TensorFlow.
* Understand image classification using neural networks.

## Topics Covered

### Neural Network Architecture

A neural network consists of:

* **Input Layer** – Receives input features.
* **Hidden Layer(s)** – Processes information through weighted connections.
* **Output Layer** – Produces the final prediction.

### Activation Functions

Activation functions help neural networks learn complex patterns by introducing non-linearity into the model.

Examples discussed:

* Sigmoid
* ReLU
* Softmax
