# Day 5 - Full Stack Development and AI-Assisted Portfolio Creation

## Overview

The fifth day of the internship focused on understanding the fundamentals of Full Stack Development, modern software development workflows, and AI-assisted coding tools. The session introduced the complete application development lifecycle, CI/CD concepts, prompt engineering, and AI-powered development environments.

## Learning Objectives

* Understand the software development lifecycle.
* Learn the basics of Full Stack Development.
* Gain awareness of CI/CD practices.
* Understand the importance of effective prompting.
* Explore AI-assisted software development tools.
* Learn how to run web applications using Node.js.

## Topics Covered

### 1. Introduction to Full Stack Development

The session began with an overview of Full Stack Development and the different stages involved in building a software application.

The development lifecycle discussed included:

* Requirement Gathering
* Design and Planning
* Development
* Testing
* Deployment
* Maintenance

The importance of collaboration between frontend, backend, and deployment teams was also discussed.

### 2. Introduction to CI/CD

A brief introduction to Continuous Integration and Continuous Deployment (CI/CD) was provided.

Key concepts covered:

* Continuous Integration
* Continuous Deployment
* Automated Testing
* Deployment Pipelines

The discussion focused on how CI/CD helps developers deliver applications efficiently and reliably.

### 3. Introduction to Prompt Engineering

The mentors introduced the concept of prompting and how AI models can generate code based on user instructions.

Topics discussed:

* Writing effective prompts
* Providing clear requirements
* Iterative prompt refinement
* AI-assisted software development

### 4. Exploring Antigravity IDE

An introduction to Antigravity IDE was provided as an AI-powered development environment.

The tool demonstrated how developers can:

* Describe requirements in natural language
* Generate application code using AI
* Modify applications through prompts
* Accelerate the development process

### 5. Portfolio Website Development

Using Antigravity IDE, a simple portfolio website was created through prompt-based development.

Activities performed:

* Defined project requirements through prompts.
* Generated the portfolio website using AI.
* Reviewed the generated code and structure.
* Understood how AI translates requirements into functional applications.

### 6. Node.js Installation and Execution

To run the generated portfolio application locally:

* Installed Node.js.
* Configured the development environment.
* Executed the application using Node.js.
* Opened and verified the portfolio website in the browser.

## Key Learnings

* Fundamentals of Full Stack Development.
* Software Development Lifecycle (SDLC).
* Basic understanding of CI/CD workflows.
* Prompt Engineering concepts.
* AI-assisted code generation.
* Working with Antigravity IDE.
* Node.js installation and setup.
* Running web applications locally.

## Outcome

Successfully learned the fundamentals of Full Stack Development, explored AI-assisted software development using Antigravity IDE, created a portfolio website through prompt-based development, and executed the application locally using Node.js.
