## Topics Covered

### Regression
Learned how regression algorithms are used to predict continuous numerical values.

Real-world examples discussed:
- House price prediction
- Student marks prediction
- Sales forecasting
- Temperature prediction

### Classification
Learned how classification algorithms are used to predict categorical outcomes.

Real-world examples discussed:
- Spam email detection
- Disease prediction
- Pass/Fail prediction
- Customer churn prediction
- Survival prediction
