### Day 4 - Convolutional Neural Networks and Model Comparison

- Learned the fundamentals of Convolutional Neural Networks (CNNs) and their architecture.
- Explored key CNN components such as convolution layers, pooling layers, activation functions, and fully connected layers.
- Discussed popular image classification models including CNN, MobileNetV2, ResNet50, and EfficientNetB0.
- Implemented an image classification model using Hugging Face and MobileNetV2.
- Trained and evaluated multiple models on the Intel Image Classification Dataset.
- Compared model performance based on classification accuracy.

#### Intel Image Classification Dataset Results

| Model | Accuracy |
|---------|----------|
| MobileNetV2 | 77.45% |
| CNN | 58.55% |
| ResNet50 | 22.15% |
| EfficientNetB0 | 10.80% |

- MobileNetV2 achieved the highest accuracy and was identified as the best-performing model for the dataset.

#### Flowers Dataset Results

| Model | Accuracy |
|---------|----------|
| MobileNetV2 | 90.19% |
| CNN | 66.76% |
| ResNet50 | 35.69% |
| EfficientNetB0 | 21.66% |

- Evaluated the same models on a Flowers Dataset and observed consistent performance trends.
- MobileNetV2 again achieved the best classification accuracy among all tested models.
